## Keyboard Glockenspiel

I've printed all of the remaining hammers and have super-glued a bunch of the ball bearings on to them: https://img.incoherency.co.uk/6633

These are glued using a drop of medium-viscosity super glue, I used it quite sparingly so won't be surprised if some of them come off, I can just re-glue them
as and when that happens.

I've also glued the black keytops on, using Stormsure flexible repair adhesive. I haven't used this before, I chose it over superglue because I don't want white
marks around the edges of the join. Again I was quite sparing with it but the joint for the keytops is basically only loaded in compression so the glue doesn't even have
to do anything really.

Alright, first tune! Mary Had a Little Lamb: https://www.youtube.com/watch?v=R_7Fbg969f4

The key activation force is about "4x M6 nuts", which I weighed at 12 grams, so it wants about 4x as much activation force, which I can achieve by adding
counterweight to the backs of the keys.
